# BoneAnimatedMesh-WebRender

**[🚀 Live Demo → woodmaniac13.github.io/BoneAnimatedMesh-WebRender](https://woodmaniac13.github.io/BoneAnimatedMesh-WebRender/)**

A single-page WebGL viewer for bone-animated 3D meshes, built with [Three.js](https://threejs.org/).

## Features

- Procedural 12-bone humanoid character with weighted vertex skinning
- Animated spine S-curve, arm swings, and wrist curl
- Skeleton overlay, wireframe mode, speed & amplitude controls
- Drag-and-drop **.glb / .gltf** file loader with AnimationMixer support
- Orbit, zoom, and pan with mouse / trackpad

## How to enable GitHub Pages

1. Go to **Settings → Pages** in this repository
2. Set **Source** to **GitHub Actions**
3. The workflow runs automatically on every push to `main`

## Local usage

Open `index.html` directly in any modern browser — no build step needed.
